# 💫 About Me:

## Hey, I'm David 👋

Frontend developer specializing in React and TypeScript with 3 years of hands-on experience building responsive, high-performance web applications. I'm passionate about creating clean, user-focused interfaces that combine smooth animations, intuitive interactions, and optimized performance.

I work across the full development lifecycle—from collaborating with designers on pixel-perfect Figma implementations to deploying production-ready applications on Vercel and AWS. My approach emphasizes maintainable code, accessibility, and seamless experiences across all devices and screen sizes.

Currently balancing my Mechanical Engineering studies at OAU with active freelance work, building real solutions for startups and clients. I thrive on turning complex problems into simple, elegant interfaces that users actually enjoy.

When I'm not coding, you'll find me playing keyboard at church, exploring new animation libraries, or refining my craft through continuous learning.
